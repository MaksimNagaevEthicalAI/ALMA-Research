## Purpose

ALMA AI is an independent ethical AI research initiative exploring whether artificial intelligence can support long-term psychological compatibility awareness before emotional attachment forms.

The project focuses on:
- reflective self-understanding
- compatibility awareness
- long-term relational wellbeing
- ethical human-centered AI systems
